Ajuda

Você deseja ajuda online para qual programa?

Navegador de modo de linha
Usando o navegador de modo de linha, selecione uma opção com um número e pressione o botão de Retorno
Viola
Para ajudar no Viola, basta pressionar o ícone de ponto de interrogação (?) próximo ao canto superior direito.
Próxima Etapa
Para selecionar a ajuda com este navegador, use a Ajuda do menu Info ou selecione esta opção com o mouse (duplo clique).
Mídias WWWW
Para obter ajuda neste navegador, selecione esta opção com o mouse.
